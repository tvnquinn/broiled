BROiled — Phase 0 wireframes, v2

Dark-first countdown, per-day scheduling, roasts via notification

Dark is now the actual default, not a media-query fallback — every screen (including onboarding and settings, which were still light before) uses the same near-black gray, and the countdown numbers match the logo's exact orange. Onboarding supports a different deadline per active day instead of one shared time.
